# Google Play Store Listing — 小爪 (LilClaw)

## App Name
小爪 — AI 口袋助手

## Short Description (80 chars max)
把 AI 装进口袋。本地运行，隐私优先，无需服务器，开箱即用的智能助手。

## Full Description (4000 chars max)
小爪是一款将 AI 助手完整运行在你手机上的应用。无需额外服务器，无需复杂配置——安装后输入你的 AI 服务 API Key，即可开始对话。

🐾 核心特性

• 本地运行 — AI 引擎完全在手机上运行，对话数据不经过第三方服务器
• 隐私优先 — 你的聊天记录只存在你的手机里
• 多模型支持 — 支持 DeepSeek、OpenAI、Anthropic、Google Gemini 等主流 AI 服务
• 语音输入 — 说出你的问题，小爪帮你解答
• 拍照提问 — 拍照或从相册选图，让 AI 看图回答
• 智能工具 — 内置搜索、写作、翻译、健康咨询等常用功能
• 离线消息 — 断网时消息自动缓存，联网后自动发送

🎯 适合谁

• 想体验 AI 但不想折腾技术的用户
• 注重隐私、不希望数据上传云端的用户
• 需要一个随身 AI 助手的任何人

📱 使用方式

1. 安装小爪
2. 首次启动会自动配置环境（约 1 分钟）
3. 在设置中填入你的 AI 服务 API Key
4. 开始和小爪聊天！

⚡ 技术说明

小爪基于 OpenClaw 开源项目构建，在手机上运行完整的 AI Gateway。支持 ARM64 设备，Android 8.0 及以上版本。
