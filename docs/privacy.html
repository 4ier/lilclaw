小爪 (LilClaw) 隐私政策

生效日期：2026 年 2 月 24 日

概述

小爪是一款本地运行的 AI 助手应用。我们高度重视用户隐私。本应用的核心设计原则是：你的数据留在你的设备上。

数据收集

小爪 不收集、不上传、不存储 任何用户个人数据到我们的服务器。我们没有服务器。

本地数据

以下数据仅存储在你的设备本地：

  • 聊天记录
  • 应用设置（包括你配置的 API Key）
  • 应用运行日志

这些数据不会被传输到我们或任何第三方。卸载应用将永久删除所有本地数据。

第三方 AI 服务

小爪需要你提供第三方 AI 服务的 API Key（如 DeepSeek、OpenAI 等）才能工作。当你发送消息时，你的消息内容会直接发送到你选择的 AI 服务提供商。这些通信：

  • 由你的设备直接发送到 AI 服务商
  • 不经过我们的任何中间服务器
  • 受该 AI 服务商自己的隐私政策约束

权限说明

  • 网络访问 — 连接你配置的 AI 服务
  • 相机 — 拍照提问功能（仅在你主动使用时）
  • 麦克风 — 语音输入功能（仅在你主动使用时）
  • 前台服务 — 保持 AI 引擎在后台运行

应用更新

小爪会在启动时检查 GitHub 上的版本信息以提供组件更新。此过程仅下载公开的版本清单文件，不发送任何用户数据。

儿童隐私

小爪不面向 13 岁以下儿童，也不会有意收集儿童的个人信息。

变更

如果本隐私政策发生变更，我们会在此页面更新。

联系方式

如有任何关于隐私的问题，请通过 GitHub Issues 联系我们：github.com/4ier/lilclaw/issues
